**Task 1:**

**🚀 Task Overview**

I built this static resume website to:

Showcase my professional details in a visually appealing format.

Demonstrate my core frontend skills (semantic HTML, modern CSS).

Provide an interactive alternative to traditional PDF resumes.

**🧰 Technologies Used**

HTML5: Structured, semantic markup.

CSS3: Flexbox, Grid, animations, and responsive design.

✨ Key Features
✔ Clean, Modern Layout – Easy-to-read sections with balanced typography.
✔ Fully Responsive – Adapts seamlessly to mobiles, tablets, and desktops.
✔ No Frameworks – Pure vanilla CSS (no Bootstrap/Tailwind).
✔ Lightweight – Fast loading with minimal dependencies.

**Portfolio Website - CSS Structure (Part 2)**

**🌟 Task Overview**

This repository contains the CSS styling implementation for my professional portfolio website. Building upon the HTML structure from Part 1, this phase adds visual appeal, responsive design, and modern layout techniques.

**🎯 Key Features**

Mobile-first responsive design with media queries

Flexbox and Grid for modern layouts

Consistent color scheme and typography

Interactive elements with hover effects

Polished UI components (navigation, cards, footer)

**📱 Responsive Breakpoints**

Screen Size	Layout Adaptations
< 576px	Single column, compact spacing
576-768px	Adjusted typography sizes
768-992px	2-column project grid
> 992px	3-column grid, expanded spacing

**🖌️ Design System**

Colors:

Primary: #3498db

Secondary: #2ecc71

Dark: #2c3e50

Light: #ecf0f1

Typography:

Headings: 'Montserrat', sans-serif

Body: 'Roboto', sans-serif

Spacing:

Base unit: 1rem

Section padding: 3rem

**🔧 What I Implemented**

Mobile-first CSS architecture

Flexbox navigation with smooth hover transitions

CSS Grid for project gallery layout

Responsive typography scaling

Consistent spacing system

Cross-browser compatible styles
